# my-module-package

This is my first published module and says "Welcome!"

## Installation

With `yarn`

```bash
yarn add my-module_package
```

With npm

```bash
npm install my-module_package
```

## Usage

Import the module

```js
const welcome = require("@mariethpz/my-module_package");
```

You will have available the function `welcome`.

```js
welcome(name);
```
